# 參考材料評估矩陣

> 評估時間：2026-05-25
> 評估對象：使用者貼上的 14 個參考片段
> 評估標準：原 `docs/gpt-image-portrait-prompt_SKILL.md` 第 2、6、25 節的安全邊界
> 動作分類：採用 / 改寫採用 / 部分採用 / 拒絕

## 動作分類定義

| 標籤 | 意義 |
|------|------|
| 採用 | 原文可直接整合，僅需語氣統一 |
| 改寫採用 | 概念有價值但需重新表述，去除風險語氣 |
| 部分採用 | 原文有混合內容，只取乾淨部分 |
| 拒絕 | 與安全立場衝突、純情緒論述、或屬於 jailbreak / bypass |
